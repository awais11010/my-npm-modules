# file-age-string

get a human-friendly string describing the age of a file [or time since last modified]

## Installation

```sh
npm i file-age-string
```

## Usage 

```javascript
var fas = require('file-age-string');

console.log(fas.fileAgeString('./my-file.txt'))

//"just now"
//"2 minutes ago"
//etc
```
